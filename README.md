#  Trouve ton artisan

Plateforme web régionale permettant de consulter et rechercher des artisans en Auvergne-Rhône-Alpes.

##  Objectif du projet

Développer une application web complète avec :

- Un frontend en React
- Un backend en Node.js / Express
- Une base de données MySQL
- Une API REST
- Une recherche dynamique des artisans
- Une page 404
- Un footer conforme aux exigences
- Le respect de la charte graphique régionale

---

##  Technologies utilisées

Frontend :
- React
- React Router
- Bootstrap
- Axios

Backend :
- Node.js
- Express
- Sequelize (ORM)

Base de données :
- MySQL

---

##  Fonctionnalités principales

- Affichage des artisans par catégorie
- Fiche détaillée d’un artisan
- Barre de recherche par nom
- Formulaire de contact
- Page 404
- Pages légales (structure en place)
